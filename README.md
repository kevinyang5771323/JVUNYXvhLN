# 车辆信息管理系统 python467

## 项目概述

车辆信息管理系统是一款基于Python3.6.5、pyqt5和MySQL开发的软件，主要针对车辆及用户信息进行管理。系统具备完善的登录注册机制，以及丰富的数据管理功能。

## 技术栈

- 后端：Python3.6.5
- 前端：pyqt5
- 数据库：MySQL

## 功能模块

### 用户管理

- 登录注册：区分管理员和普通用户，管理员可注册新用户，确保用户名不重复。

### 数据管理

- 用户信息管理：管理员专属，提供数据的增删改查功能。
- 驾驶人信息管理：支持数据的增删改查，以身份证号作为唯一标识。
- 机动车信息管理：实现数据的增删改查，以发动机号作为唯一标识。

## 系统角色

- 管理员：负责用户注册和用户信息管理。
- 普通用户：进行登录并使用系统。

## 运行环境

- 操作系统：兼容Windows、Linux等平台。
- 环境依赖：Python3.6.5及以上版本，MySQL数据库，pyqt5库。

## 部署步骤

1. 安装Python3.6.5及以上版本。
2. 安装MySQL数据库，并创建对应的数据表。
3. 安装pyqt5库。
4. 部署源代码。
5. 运行系统。

## 目录结构

```
- main.py          # 主程序入口
- models.py        # 数据模型
- views.py         # 视图层
- controllers.py   # 控制器层
- resources/       # 资源文件
- sql/             # SQL脚本
- tests/           # 测试代码
```
